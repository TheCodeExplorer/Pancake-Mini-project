# 🥞 Pancake Master

Pancake Master is a sleek, modern single-page web application designed to help users discover, search, and master the art of pancake making. From classic buttermilk pancakes to savory South Indian Uttapam, the application provides detailed recipes with interactive step-by-step guides.

## ✨ Features

- **🔒 Mock Authentication**: Interactive login and registration system to customize your cooking journey.
- **🔍 Smart Recipe Search**: Easily search and filter through various pancake recipes.
- **📋 Interactive Ingredients Checklist**: Keep track of ingredients you've prepared with interactive checkable lists.
- **👩‍🍳 Step-by-Step Instructions**: View dynamic instructions for each recipe with a progress tracker/completion flow.
- **📱 Fully Responsive Design**: Beautifully styled using vanilla CSS, with transitions and responsive layouts for mobile and desktop screens.

## 🛠️ Built With

- **HTML5**: Semantic markup for clean structure and SEO optimization.
- **CSS3**: Custom design with Google Fonts (Inter), custom transitions, grid/flexbox layouts, and premium UI components.
- **JavaScript (ES6+)**: Interactive client-side logic, recipe filtering, form validation, and state management.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your web browser:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/TheCodeExplorer/Pancake-Mini-project.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Pancake-Mini-project
   ```
3. **Open the project**:
   - Double-click the `index.html` file, or
   - Use a development server such as VS Code's Live Server extension.
